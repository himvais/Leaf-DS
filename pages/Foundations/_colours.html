Colours

Our colour system provides the flexibility of customization for various products while maintaining consistency and recognition effects for interfaces.

Colours API

Use the following class system to get various background colours on any element:

Class Name Effects
.bg-transparent Transparent Background
.bg-white White Background
.bg-black Black Background
.bg-light-100 Background with the colour theme-100
.bg-light-200 Background with the colour theme-200
.bg-primary Background with the colour primary brand colour or theme-500
.bg-dark-100 Background with the colour theme-600
.bg-dark-200 Background with the colour theme-700
.bg-success Background with the colour success-light or green-200
.bg-warning Background with the colour warning-light or yellow-200
.bg-error Background with the colour error-light or red-200
All the classes can be user with hover:classname to achieve the same effect on hover
