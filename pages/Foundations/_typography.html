Text Sizing

text-* family of classes can be used to assign font-size to any text element, it consist of following classes and their respective effects:

classname font-size line-height usage guide preview
.text-xs 12px 14px Super small text, footnotes, pill text, advised to used witd large font weights.

.text-sm 14px 16px Small text, subtext, not optimum for longer reading experience, good for short messages.

sample
.text-md 16px 24px Default paragraph text size

.text-lg 18px 24px Large paragraph text, stronger paragraphs, best for explaining or high empesis texts.

sample
.text-h5 20px 24px Smallest headings or h4 size headings, great for section labels or low level headings.

Text Weight

text-weight-* family of classes provide the font-weight access with light, regular, semibold and bold options:

classname font-weight usage guide preview
.text-weight-light 200 Should only be used with larger text size and for leading or helping texts.

Sample
.text-weight-normal 400 default font-weight, should be used for all regular texts.

Sample
.text-weight-semibold 600 Semi-bold text, to be used with most of the headings and emphesized texts.

Sample
.text-weight-bold 800 bold text, should be used with consideration and to put emphesis on words in longer textforms.
