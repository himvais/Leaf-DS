Layouts

Layout system in leaf-foundation allows you to create various layouts using CSS grids and Flex.

_layout.scss

Display API

Display classes follow .d-* syntax, The following classes can be used for changing the css display property of any element.


class property
.d-block display:block
.d-inline display:inline
.d-inline-block display:inline-block
.d-flex display:flex
.d-inline-flex display:inline-flex
.d-grid display:grid
.d-inline-grid display:inline-grid
.d-hidden display:hidden

Flow API

The Flow API allows controlling how the content will flow inside a parent grid or flex container. It works with both grid and flex based layouts.

Class formula: .flow-[x|y|xy]-[start|end|center|between|around|evenly]


class effect CSS
.flow-x-start horizontally, left to right justify-content: flex-start
.flow-x-end horzontally, right to left justify-content: flex-end
.flow-x-center horizontally, aligned in center justify-content: center
.flow-x-between horizontally, the space between content splits evenly justify-content: space-between
.flow-x-around horizontally, the space around content is distributed evently justify-content: space-around
.flow-x-evenly horizontally, the content is spread evenly in container justify-content: space-evenly
Replace x with "y" for vertical and "xy" for both vertical and horizontal results.

Content API

When using a grid or flex layout, the content api helps with setting the behaviour of the content within it's row/column or container

Formula: .content-[x|y|xy]-[start|end|center|stretch]

class effect CSS
.content-x-start Items are placed left to right justify-items: start
.content-x-end Items are placed right to left justify-items: end
.content-x-center Items are placed at the center of parent justify-items: center
.content-x-stretch Items are streched to occupy full width of parent Justify-items: stretch
Replace x with "y" for vertical and "xy" for both vertical & horizontal results.
