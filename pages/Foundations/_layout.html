Layouts

Layout system in leaf-foundation allows you to create various layouts using CSS grids and Flex.

_layout.scss

Display API

Display classes follow .d-* syntax, The following classes can be used for changing the css display property of any element.


class property
.d-block

display:block
.d-inline

display:inline
.d-inline-block

display:inline-block
.d-flex

display:flex
.d-inline-flex

display:inline-flex
.d-grid

display:grid
.d-inline-grid

display:inline-grid
.d-hidden

display:hidden

Flex classes

Flex classes follow .f-* syntax, The following classes can be used for changing the flex property of any element.


class property
.f-row

flex-direction:row
.f-row-reverse

flex-direction: row-reverse
.f-col

flex-direction: column
.f-col-reverse

flex-direction: column-reverse

Flow API

The Flow API allows controlling how the content will flow inside a parent grid or flex container. It works with both grid and flex based layouts.

Class formula: .flow-[x|y|xy]-[start|end|center|between|around|evenly]


class effect CSS
.flow-primary-start left to right or top to bottom based on primary axis

justify-content: flex-start
.flow-primary-end right to left or bottom to top based on primary axis

justify-content: flex-end
.flow-primary-center aligned in center horizontally or vertically based on primary axis

justify-content: center
.flow-primary-between the space between content splits evenly

justify-content: space-between
.flow-primary-around the space around content is distributed evently

justify-content: space-around
.flow-primary-evenly the content is spread evenly in container

justify-content: space-evenly
Replace primary with "secondary" for setting flow of secondary axis and "xy" for both vertical and horizontal results.

Content API

When using a grid or flex layout, the content api helps with setting the behaviour of the content within it's row/column or container

Formula: .content-[x|y|xy]-[start|end|center|stretch]

class effect CSS
.content-primary-start left to right or top to bottom based on flex direction

justify-items: start
.content-primary-end right to left or bottom to top based on flex direction

justify-items: end
.content-primary-center Items are placed at the center of parent

justify-items: center
.content-primary-stretch Items are streched to occupy full width of parent

Justify-items: stretch
Replace primary with "secondary" for setting flow of secondary axis and "xy" for both vertical & horizontal results.
