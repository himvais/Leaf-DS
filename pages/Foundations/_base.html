Foundations

A utility-first CSS library for facilitating front-end development with Leaf Design System.

Download Foundation

How to use

Foundations.css is available as a standalone file that you can include in your projects. It exposes a CSS api (collection of classes ready to use in your HTML) which you can use to get the desired effects.

You should include a CSS reset file before including the foundation.css to ensure a consistent cross-browser experience.
Once included, you're ready to use the foundations classes in your front-end code, please refer to the sections on the right to understand available classes.

What is utility first?

Utility first css frameworks provide a low-level utility classes to build out custom designs within HTML with pre-defined standardization applied.

Utility classes helps you quickly apply colour, borders and spacing and so much more.

Theming

There are a few things you can customize in Foundations, mainly the brand colour palette. You can provide a new set of brand colours in theme.scss and compile the foundation.scss again with any SCSS tool to generate a customised version of foundation.
