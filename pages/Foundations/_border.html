Borders

standard definitions for borders, dividers and artifact lines in leaf design system.

_border.scss

Border API

Border API exposes an eloborated class system to create any border system you want within the Leaf design system specifications.


Border Width

border-* family of classes can be used to assign borders to any text element, it consist of following classes and their respective effects:

classname border-width preview
.border-100 1px
.border-200 2px
.border-300 3px
.border-400 4px
.border-500 5px

Border Radius

border-radius-* family of classes can be used to assign border radius to any text element, it consist of following classes and their respective effects:

classname border-radius preview
.border-radius-small 2px
.border-radius-medium 4px
.border-radius-large 8px
.border-radius-round 999px

Border Style

border-style* family of classes can be used to assign border style to any text element, it consist of following classes and their respective effects:

classname border-style preview
.border-style-solid solid
.border-style-dashed dashed
.border-style-dotted dotted
.border-style-hidden hidden
.border-style-none none

Border Color

border-color-* family of classes can be used to assign border color to any text element, it consist of following classes and their respective effects:

classname border-color preview
.border-color-default default
.border-color-dark dark
.border-color-brand brand
